# Streamlit Demo: The Fashion MNIST from TensorFlow

This project demonstrates the [Fashion MNIST dataset](https://github.com/tensorflow/docs/blob/master/site/en/tutorials/keras/classification.ipynb) into an interactive [Streamlit](https://streamlit.io) app.

The complete demo is [MNIST with Streamlit](https://https://github.com/taeokimeng) and illustrates all the major building blocks of Streamlit.
